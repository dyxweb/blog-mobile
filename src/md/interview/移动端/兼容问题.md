### ios日期转换NaN的问题
```
// 将日期字符串的格式符号替换成'/'
'yyyy-MM-dd'.replace(/-/g, '/')
```
### iOS上拉边界下拉出现空白
- 手指按住屏幕下拉屏幕顶部会多出一块白色区域。手指按住屏幕上拉底部多出一块白色区域。
- 在iOS中手指按住屏幕上下拖动，会触发touchmove事件。这个事件触发的对象是整个webview容器，容器自然会被拖动，剩下的部分会成空白。
```
document.body.addEventListener(
  'touchmove',
  function(e) {
    if (e._isScroller) {
      return;
    }
    // 阻止默认事件
    e.preventDefault()
  },
  {
    passive: false
  }
)
```
### ios下点击input页面会放大
- html添加禁止缩放的meta
- 添加样式
```
input {
  font-size: 16px;
}
```
